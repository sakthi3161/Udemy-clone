# 💻 Technology Cards Website

A simple and responsive website built using **HTML** and **CSS** that showcases different technology cards with images and descriptions. This project is designed to practice HTML structure, CSS styling, and responsive layouts.

![Project Preview](images/html.jpg)

## 🚀 Features

- ✅ Clean and modern UI
- ✅ Responsive card layout
- ✅ HTML5 semantic structure
- ✅ CSS styling and hover effects
- ✅ Easy to customize

- HTML5
- CSS3
